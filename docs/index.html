One lifecycle map, many specialist skills.

The package is designed for handoffs: discover, write, verify, submit, respond, publish, and present. Use one skill directly or let an agent orchestrate the sequence.

Common workflows.

These are the high-signal paths users can copy into an agent or follow step by step.

New paper paper-profile -> literature-review -> write-abstract -> draft-related-work -> polish-prose
Before submission parse-cfp -> tailor-to-venue -> anonymize-paper -> preflight-check
Quality pass verify-citations -> check-originality -> benchmark-paper -> simulate-reviewers -> assess-paper
Accepted paper prepare-camera-ready -> make-slides -> write-talk-script -> rehearse-qa -> make-poster
