AI agent skills for the full research-paper lifecycle.

Built for the work around the paper.

Each folder is a portable Agent Skills task with trigger rules, guardrails, references, and deterministic helpers where repeatability matters.

Discover

Find papers, fetch legal open-access copies, study exemplars, and build grounded literature reviews.

Write

Draft abstracts, related work, surveys, prose edits, structure repairs, and figure/table polish.

Verify

Catch fabricated citations, unsupported claims, originality risks, and reproducibility gaps.

Submit

Parse CFPs, tailor drafts to venues, anonymize, check page limits, and simulate reviewers.

Respond

Triage reviews and draft author responses for rebuttal, revision, and journal workflows.

Publish and present

Prepare camera-ready packages, artifacts, slides, talk scripts, Q&A practice, and posters.

One lifecycle map, many specialist skills.

Use one skill directly or let an agent orchestrate the sequence with checkpoints and live source verification.

Common research workflows.

Plain-language requests trigger the right skill and keep the author in control.

New paper: paper-profile -> literature-review -> write-abstract -> draft-related-work -> polish-prose Before submit: parse-cfp -> tailor-to-venue -> anonymize-paper -> preflight-check Survey draft: find-papers -> draft-survey -> verify-citations -> polish-prose Accepted: prepare-camera-ready -> make-slides -> write-talk-script -> rehearse-qa -> make-poster
